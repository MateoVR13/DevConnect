Tipos de datos en Python

Los tipos de datos en Python son una forma de clasificar los valores que pueden almacenarse en variables. Python tiene una amplia variedad de tipos de datos, incluyendo tipos primitivos, tipos compuestos, y tipos definidos por el usuario.

Tipos primitivos:

Los tipos primitivos son los tipos de datos más básicos de Python. Incluyen los siguientes:

Flotante: Los flotantes son números con decimales. Por ejemplo, 1.0, 2.3, 3.14159, etc.

Cadenas: Las cadenas son secuencias de caracteres. Por ejemplo, "Hola mundo", "123456", etc.

Booleanos: Los booleanos son valores que pueden ser True o False.

Nulo: El nulo es un valor especial que indica que una variable no tiene un valor asignado.


Ejemplos: 

                      
        # Entero
        numero = 10
        
        # Flotante
        decimal = 1.23456
        
        # Cadena
        texto = "Hola mundo"
        
        # Booleano
        verdadero = True
        falso = False
        
        # Nulo
        nulo = None

Tipos compuestos:

Los tipos compuestos son tipos de datos que pueden almacenar más de un valor. Incluyen los siguientes:

Flotante: Las listas son secuencias de valores.

Cadenas: Las tuplas son secuencias de valores inmutables.

Booleanos: Los diccionarios son colecciones de pares clave-valor.


Ejemplos: 

                    
        # Lista
        numeros = [1, 2, 3, 4, 5]
        
        # Tupla
        nombres = ("Juan", "Pedro", "María")
        
        # Diccionario
        persona = {"nombre": "Juan", "edad": 30}
Variables en Python

Una variable es el nombre que se le da a una ubicación de memoria. Una variable de Python que contiene valores también se conoce como identificador. Dado que Python es un lenguaje de inferencia lo suficientemente inteligente como para determinar el tipo de una variable, no necesitamos especificar su tipo en Python.Los nombres de las variables deben comenzar con una letra o un guión bajo, pero pueden ser un grupo de letras y dígitos. El nombre de la variable debe escribirse en minúsculas. "Tanto Variable" como "variable" son variables distintas.
